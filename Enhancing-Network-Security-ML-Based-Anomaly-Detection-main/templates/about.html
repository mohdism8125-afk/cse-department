About This Project

Enhancing Network Security: ML-Based Anomaly Detection

This system uses Machine Learning to analyze network traffic parameters and detect potential cyber attacks in real-time. By examining features such as packet characteristics, anomaly scores, protocol types, and security indicators, the Random Forest classifier can identify 6 types of network threats.

The model was trained on synthetic network traffic data that mimics real-world attack patterns — DDoS attacks show high anomaly scores and critical severity, malware traffic carries IoC (Indicators of Compromise), and intrusion attempts target SSH/FTP services.

Attack Types Detected
AttackDescriptionKey Indicators
DDoS Distributed Denial of Service — floods the target with traffic High anomaly score, Critical severity, IDS alerts triggered
Intrusion Unauthorized access attempts via SSH, FTP, or other services SSH/FTP traffic, elevated anomaly score, High severity
Malware Malicious software communicating with command-and-control servers IoC Detected, moderate-high anomaly score
Phishing Social engineering attacks disguised as legitimate traffic Suspicious activity patterns, varied anomaly scores
Ransomware Encryption-based extortion attacks targeting data High anomaly, suspicious indicators, elevated severity
Normal Legitimate network traffic — no threat detected Low anomaly (<0.3), Low severity, No alerts, No IoC
Technology Stack

  • Frontend: Bootstrap 5, Chart.js
  • Database: SQLite
  • Security: Werkzeug password hashing
  • Encoding: LabelEncoder (10 features)
